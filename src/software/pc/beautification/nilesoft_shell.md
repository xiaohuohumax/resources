## 简介

Nilesoft Shell 是一个上下文菜单定制器，可让您手动选择要集成到 Windows 文件资源管理器上下文菜单中的项目，创建自定义命令以访问您喜爱的所有网页、文件和文件夹，以及直接从上下文菜单启动任何应用程序。
