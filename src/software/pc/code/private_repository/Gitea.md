## 简介

Gitea 是一个无痛的自托管一体化软件开发服务，包括 Git 托管、代码审查、团队协作、包注册和 CI/CD。它在 MIT 许可证下开源。Gitea 被设计为轻量级、易于使用且高度可定制，非常适合小型团队和大型组织使用。
