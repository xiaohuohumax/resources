## 简介

Harbor 是一个开源的注册中心，通过策略和基于角色的访问控制来确保工件的安全，确保镜像经过扫描且没有漏洞，并对镜像进行可信签名。Harbor 提供合规性、性能和互操作性，帮助您在 Kubernetes 和 Docker 等云原生计算平台上始终如一且安全地管理工件。
