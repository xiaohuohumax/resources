## 简介

Bookmarks 是一个 Visual Studio Code 扩展，可以帮助您轻松快速地在代码中导航，方便地在重要位置之间移动，无需再搜索代码。它支持一系列选择命令，允许您选择书签行及其之间的区域，非常适合用于日志文件分析。
