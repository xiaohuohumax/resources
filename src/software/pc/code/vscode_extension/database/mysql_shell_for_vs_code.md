## 简介

MySQL Shell for VS Code 插件是一个强大的工具，旨在将 MySQL Shell 的功能直接集成到 VS Code 编辑器中。该插件为开发者和数据库管理员提供了一个现代化的环境，使其能够高效地与 MySQL 数据库进行交互。
