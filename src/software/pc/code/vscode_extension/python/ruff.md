## 简介

这是一种用 Rust 编写的极快的 Python linter 和代码格式化程序。Ruff 可用于替换 Flake8（以及数十个插件）、Black、isort、pyupgrade 等，同时执行速度比任何单个工具快数十倍或数百倍。
