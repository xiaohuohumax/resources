## 简介

‌UniGetUI ‌是一个为 Windows 上的最流行的命令行包管理器提供直观图形用户界面的工具。它支持 Winget、Scoop、Chocolatey、Pip、Npm、.NET Tool 和 PowerShell Gallery 等包管理器，使得用户可以轻松下载、安装、更新和卸载软件‌。
